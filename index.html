About

I build the API and the app that calls it.

At Xmplar Management Solutions I work across three tracks simultaneously — Spring Boot APIs on the backend, Android apps in Java on the mobile side, and Flutter for cross-platform clients. The Android and backend work are closely connected: the mobile apps I build consume the same APIs I design on the server, so I understand both sides of every integration.

The backend goes beyond standard CRUD — I've built OPC UA pipelines that pull telemetry from factory-floor hardware and expose it through REST, done R&D on TimescaleDB to bring query latency down on time-series data, and used Cytoscape.js to produce architecture flows that give stakeholders clarity without documentation overload.

I started building Android Java at Priga Solutions — Firebase, Google Maps, navigation features — which gave me a solid mobile foundation that carries directly into the Android work I do at Xmplar today. I document what I learn across all three stacks on Medium.
